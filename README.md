# kingpin_nslookup
根据nslookup查询域名对应ip列表 记录延时

#### 功能:
- 1 多域名支持
- 2 可添加固定ip
- 3 信息存csv/sqlite/postgres
- [remove] :fearful:
    - 4 支持写/etc/host
    - 5 时间校队, 邮件通知 服务器时间与本地时间差
